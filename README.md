# 家庭学習プリント生成｜学習プリント自動作成ツール

## 📌 アプリ概要

家庭学習や個別学習向けに、学習段階に合わせたプリントを素早く・簡単に作れる Web アプリです。  
プリントの内容・レベルを選ぶだけで問題を自動生成し、A4 サイズで表示・印刷・PDF 保存ができます。

---

## ✅ 実装済み機能

### 🔷 STEP 1 — プリント内容選択（7モード）
| 選択肢 | 内容 |
|--------|------|
| 助詞 | は・を・に・が など |
| 五十音 | あ〜ん の書き練習 |
| めいろ／並び替え／文章問題／ひらがな迷路 | 各モードの出題形式に従う |
| カスタム問題（有料） | 自分で入力した単語でなぞり・視写 |

### 🔷 STEP 2 — レベル選択（3段階）
| レベル | 形式 |
|--------|------|
| 🌱 初級 | なぞり書き（文字を薄く表示） |
| 🌼 中級 | 選択問題（3択） |
| 🌟 上級 | 記述問題（自由記述） |

### 🔷 STEP 3 — プリント設定
- 問題数：4 / 6 / 8 / 10問
- 名前欄：あり / なし
- 日付欄：あり / なし

### 🔷 プリント生成・出力
- ボタン1クリックでランダム問題を自動生成
- A4サイズのプレビュー表示
- 「印刷ボタン」：ブラウザの印刷ダイアログを開く
- 「PDF保存ボタン」：案内後に印刷ダイアログを開き、ブラウザの「PDFに保存」で保存
- 「再生成ボタン」：別パターンの問題を再生成

## 📐 問題生成ルール

### 助詞
| レベル | 形式 | 例 |
|--------|------|----|
| 初級 | なぞり書き | `わたし（は）りんごをたべる` ← 「は」を薄く表示 |
| 中級 | 3択選択 | `ぼく（　）がっこうにいく` → は / を / に |
| 上級 | 自由記述 | 空欄に助詞を書く＋ヒント表示 |

### ひらがな
| レベル | 形式 |
|--------|------|
| 初級 | 行ごとのなぞり書き（あ行・か行 など） |
| 中級 | 絵文字→3択で単語を選ぶ |
| 上級 | 絵文字を見て単語を書く（マス目付き） |

## 🖥 動作環境

- Chrome / Safari / Firefox / Edge（最新版推奨）
- PC・スマートフォン・タブレット対応（レスポンシブ）
- 日本語 UI
- 外部依存：Google Fonts（Noto Sans JP）、Font Awesome（※ html2canvas / jsPDF は PDF 保存を印刷経由に変更したため未使用の場合あり）

## ⌨️ キーボードショートカット

| キー | 操作 |
|------|------|
| `Enter` | プリント生成 |
